CityScope/Fieldtrip

Fieldtrip is an app that puts you in control. Please, read the following lines to start using this service:

Edinburgh CityScope Fieldtrip
This is a mobile app that allows you collecting data for your custom surveys.
  • If you already downloaded the app on your iOS device, please visit the following link.
  • Otherwise, download the app first at the app store and visit the link afterwards.
Survey Designer
This web application permits you designing custom surveys. Please, visit the link to modify an example survey.
Survey Previewer
This web application gives you the opportunity to preview your custom surveys as they would appear on the mobile app. Please, visit the link to preview an example survey.
Records Viewer
This web app allow you to examine the data collected for an specific survey selected. Please, visit the link to view any survey record from an example survey.
